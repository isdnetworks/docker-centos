# CentOS

Customized build of CentOS.
-	Default locale: ko-KR.UTF-8
-	Default timezone: Asia/Seoul
-	EPEL GeoIP-GeoLite-data-extra installed

For more details see [centos](https://hub.docker.com/_/centos)
